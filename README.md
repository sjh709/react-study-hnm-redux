## 1. 프로젝트 소개

- h&m 쇼핑몰 사이트 입니다.
- 홈 화면을 통해 상품 전체 리스트를 볼 수 있습니다.
- 상품을 클릭하면 상품 상세 페이지로 이동합니다.
- 상품 상세 페이지는 로그인 후 볼 수 있습니다.
- 홈 화면에서 로그인 버튼을 클릭하면 로그인 화면으로 이동합니다.
- 홈 화면에서 상품을 검색할 수 있습니다.
